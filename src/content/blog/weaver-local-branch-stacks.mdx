Stacked branches are often treated as a review-platform feature. Weaver takes the opposite angle: the stack is local state first.

The CLI stores dependency relationships in `.git/weaver/`, keeps them out of the committed tree, and resolves stacks as a DAG. That makes the workflow independent of GitHub metadata, PR naming conventions, or a hosted service.

## Declare, Inspect, Then Mutate

The command set has a nice bias: inspect before changing things.

`weaver stack` and `weaver unstack` declare parent-child relationships. `weaver deps`, `weaver status`, and `weaver doctor` show the stack and diagnose broken local state. Only then do commands like `weaver update`, `weaver sync`, and `weaver compose` start moving branches around.

That order matters because stacked work is easy to make mystical. Weaver keeps the model plain: dependencies, upstream refresh, sync strategy, and integration output.

## Rebase, Merge, Or Compose

Weaver separates three jobs that often get blurred.

`weaver update` fetches upstream refs and fast-forwards selected local branches. It does not sync a branch with its stack parent.

`weaver sync` brings a stack back into dependency order. By default it uses rebase for clean stacked diffs, but `weaver sync --merge` preserves branch history when open PRs, review comments, or other consumers make rewriting awkward.

`weaver compose` builds an ephemeral integration state from several branches on top of a base. It can also create or update a named integration branch, and saved integration strategies live in `.git/weaver/integrations.yaml`.

## The Safety Model Is The Product

The README is refreshingly explicit about failure behavior. Mutating Git commands are printed before execution. Sync state is persisted before each step. `weaver abort` restores the original branch. Compose is ephemeral by default, and failures name the branch and conflicting files.

That is the right taste for a Git tool. The value is not hiding Git. The value is making the local orchestration inspectable enough that a developer can trust it.
